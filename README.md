DC CLEANER
=============

## 여러분의 흑역사를 지워줄 디씨 클리너 

# 사용 방법

# 1. python3 설치 

# 2. 커맨드라인(cmd) 혹은 powershell 실행

# 3. 패키지 설치 - requests, bs4, selenium (cmd 혹은 powershell 에 pip install requests bs4 selenium)

# 4. cmd 혹은 powershell에서 클리너가 설치된 폴더로 경로 이동 후 python dc_cleaner.py 입력

# 5. ID 및 PW 입력 후 엔터
